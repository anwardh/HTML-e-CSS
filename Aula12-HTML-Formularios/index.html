Nova Solicitação de Passaporte

Pode haver mais de um Forms por página e cada um deles poderá enviar dados de maneira separada para o Servidor.

Precisaremos indicar para onde estes dados irão (servidor, para algum outro recurso do código-fonte), através da Propriedade 'action'.

- - > É importante lhe atribuir um nome, atravé da propriedade 'name'.

- - > Todo Forms precisará de um 'método de envio', aqui, usaremos o 'get'.

IMPORTANTE

Existem duas formas de enviar os dados coletados neste Forms para o Servidor. O primeiro é o 'get' e, o segundo, o 'post'. Neste, os valores das variáveis não são apresentados na URL, tal qual o 'get', dessa forma, faz-se necessária a criação de uma pasta - Servidor - para receber no arquivo - index.html - os dados oriundos deste Forms.

Todas as alterações necessárias para isso estarão 'comentadas' neste código.
